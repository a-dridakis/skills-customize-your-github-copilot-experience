# 📘 Assignment: Python Data Structures

## 🎯 Objective

Master Python's core data structures: lists, dictionaries, sets, and tuples. You'll build a student record management system that demonstrates how to store, manipulate, and organize data efficiently using the right data structure for each task.

## 📝 Tasks

### 🛠️	Working with Lists and Tuples

#### Description
Create a program that manages a list of student records where each record is a tuple containing (student_id, name, grade). Implement functions to add students, remove students, sort by grade, and search by name or ID.

#### Requirements
Completed program should:

- Store student records as tuples with student_id, name, and grade
- Implement a function to add a new student to the list
- Implement a function to remove a student by ID
- Sort students by grade in ascending or descending order
- Search for a student by ID or name and return their record
- Display all students in a formatted table


### 🛠️	Using Dictionaries for Efficient Data Access

#### Description
Refactor your student management system to use a dictionary where student IDs are keys. Implement advanced features like grade statistics, class rosters by grade, and bulk operations. Add the ability to store additional student information (email, phone, major).

#### Requirements
Completed program should:

- Store student records in a dictionary with student_id as the key
- Add, update, and delete student records efficiently
- Calculate statistics: average grade, highest grade, lowest grade, grade distribution
- Retrieve all students in a specific grade range
- Group students by grade and display class rosters
- Implement a lookup function that takes any field (name, email) and returns matching students


### 🛠️	Sets and Advanced Data Operations

#### Description
Create a program that uses sets to find relationships between student groups. Implement features to find common students between classes, unique students across sections, and use set operations for advanced queries like finding students who are in multiple clubs.

#### Requirements
Completed program should:

- Create sets for different classes or clubs (Computer Science, Mathematics, Biology, etc.)
- Find intersection of students (who are in multiple classes)
- Find union and difference of student groups
- Identify students unique to a particular class
- Track student enrollment using sets and perform efficient membership tests
- Generate reports showing overlapping class enrollments using set operations
